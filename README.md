# predictIQ
PredictIQ is an AI-powered predictive maintenance platform with component-specific ML models for higher accuracy. It combines a Flask API, Streamlit dashboard, real-time simulation, and an AI chatbot to monitor systems, detect faults, and provide maintenance insights.
